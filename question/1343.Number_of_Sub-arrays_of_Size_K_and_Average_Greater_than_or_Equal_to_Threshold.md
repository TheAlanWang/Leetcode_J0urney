
## Method 1
### Approach: Sliding Window
### Complexity
- TC: O(n)
- SC: O(1)

```python
class Solution:
    def numOfSubarrays(self, arr: List[int], k: int, threshold: int) -> int:

        left = right =  0
        total = 0
        count = 0
        while right < len(arr):
            total += arr[right]
            
            if right - left + 1 > k: # window_size > k
                total -= arr[left]
                left += 1
            
            if right - left + 1 == k:
                if total // k >= threshold:
                    count += 1
            
            right += 1
        
        return count 
```
