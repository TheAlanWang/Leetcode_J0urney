## Question
[706. Design HashMap](https://leetcode.com/problems/design-hashmap/)

## Method
### Approach
- use `for` loop to create buckets
1. Count idx (bucket position) to find where to store key
2. Use Linked list inside each bucket to handle *collisions*
eg:
```
idx: 1 [(None, None, (1, 1, None))]
idx: 2 [(None, None, None)]
```
#### Complexity:
- Time Complexity: O(1)
- Space Complexity: O(n)
### Code
```python
class Node:
    def __init__(self, key, val, nxt):
        self.key = key
        self.val = val
        self.next = nxt

class MyHashMap:
    '''
    idx:
    0 [None, None]
    1 [None, None]
    '''
    def __init__(self, size = 1069):
        self.size = size
        self.buckets = [Node(None, None, None) for _ in range(self.size)]

    def _hash(self, key):
        size = 1069
        return key % self.size

    def put(self, key: int, value: int) -> None:
        code = self._hash(key)
        cur = self.buckets[code]
        while cur.next:
            if cur.next.key == key:
                cur.next.val = value
                return
            cur = cur.next    
            
        cur.next = Node(key, value, None)


    def get(self, key: int) -> int:
        code = self._hash(key)
        cur = self.buckets[code]
        while cur.next:
            if cur.next.key == key:
                return cur.next.val
            cur = cur.next
        return -1

    def remove(self, key: int) -> None:
        code = self._hash(key)
        cur = self.buckets[code]
        while cur.next:
            if cur.next.key == key:
                cur.next = cur.next.next
                return
            cur = cur.next
```
