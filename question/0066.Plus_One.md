## Question
- Input: `digits = [9]` -> Output:`[1,0]`
## Method 1 - Optimal
#### Complexity
- Time Complexity: O(n)
- Space Complexity: O(1)
### Code
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):

            if digits[i] < 9:
                digits[i] += 1
                return digits

            digits[i] = 0

        return [1] + digits
```

## Method 2 - carry
#### Complexity
- Time Complexity: O(n)
- Space Complexity: O(n)
### Code
```python
class Solution: 
    def plusOne(self, digits: List[int]) -> List[int]: 
        res = [] 
        carry = 1 
        for d in reversed(digits): 
            new_val = d + carry 
            res.append(new_val % 10) 
            carry = new_val // 10 
        
        if carry: 
            res.append(1) 
        
        return res[::-1]
```
