Link: [51. N-Queens](https://leetcode.com/problems/n-queens/)
## Question
- Put `n` queens on an `n x n` chessboard
- No queen can attack each other

## Method - Backtracking
> [!Key Insight]
> - Positive diagonal = r + c
> - Negative diagonal = r - c
### Approach
1. Use backtracking place one queen per row
2. Checking conflicts by using set (col_set, pos_diagonal, neg_diagonal)
3. For each row, try all column from `0` to `n-1`

#### Complexity
- Time Complexity: $O(n!)$
- Space Complexity: $O(n^2)$ -> chessboard
### Code
```python
class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        res = []
        grid = [['.'] * n for _ in range(n)]
        col_set = set()
        posDig_set = set()
        negDig_set = set()
        
        def dfs(r):
            if r == n:
                res.append(["".join(r) for r in grid])
                return
            
            for c in range(n):
                if (
                    c in col_set or
                    (r + c) in posDig_set or
                    (r - c) in negDig_set
                ):
                    continue

                col_set.add(c)
                posDig_set.add(r + c)
                negDig_set.add(r - c)
                grid[r][c] = 'Q'
                
                dfs(r + 1)
                col_set.remove(c)
                posDig_set.remove(r + c)
                negDig_set.remove(r - c)
                grid[r][c] = '.'
        
        dfs(0)
        return res

```
