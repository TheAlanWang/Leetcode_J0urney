## Question
> Given an array of integers `nums` and an integer `k`, return the total number of *subarrays* whose sum equals to `k`.
> A subarray is a contiguous **non-empty** sequence of elements within an array.
> **Example 1:**
>> **Input:** nums = `[1,1,1]`, k = 2
>> **Output:** 2
> 
> **Example 2:**
>> **Input:** nums = `[1,2,3]`, k = 3
>> **Output:** 2
### Goal
- Given an array of integers `nums` and an integer `k`, return the total number of **continuous subarrays** whose sum equals `k`.

Q: Why is this considered a prefix-sum question?
- `subarray` + `sum` + `return the total number`

## Method 1: hashmap + prefix sum
### Logic
1. A subarray `nums[l..r]` has sum `k` iff: `pref[r] - pref[l-1] = k  ⇔  pref[l-1] = pref[r] - k`
	- `prefix[i] = nums[0] + nums[1] + ... + nums[i]`
	- Subarray: `nums[i:j] = prefix[j] - prefix[i-1]`
2. Use **hashmap** to record how many times a given **prefix sum** has appeared.

#### Complexity analysis
- Time Complexity: O(n)
- Space Complexity: O(n)
### Code
```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        res = 0
        prefix_map = {0: 1} # prefix sum 0 has occured 1 time
        curr_prefix_sum = 0
        
        for num in nums:
            curr_prefix_sum += num
            diff = curr_prefix_sum - k
            if diff in prefix_map:
                res += prefix_map[diff]
        
            freq = prefix_map.get(curr_prefix_sum, 0)
            prefix_map[curr_prefix_sum] = freq + 1
        return res
```


## Method 2: Brute force
#### Complexity analysis
- Time Complexity: $O(n^2)$ - Time limit Exceed
- Space Complexity: O(1)
```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        n = len(nums)
        count = 0
        for i in range(n):
            total = 0
            for j in range(i, n):
                total += nums[j]
                if total == k:
                    count += 1
        
        return count
```
